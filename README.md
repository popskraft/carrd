# Carrd.co Customization

Папка для различных кодов и кастомизаций конструктора CMS [Carrd.co](https://carrd.co).

## Структура проекта

- `css/` - Пользовательские CSS стили
- `js/` - JavaScript код для функциональности
- `html/` - HTML шаблоны и фрагменты
- `docs/` - Документация и примеры

## Использование

Коды из этого репозитория можно использовать для расширения функциональности Carrd сайтов через:
- Встраивание кастомного CSS
- Добавление JavaScript функций
- Создание переиспользуемых компонентов
