# Carrd.co Customization

Репозиторий для различных кодов и кастомизаций конструктора CMS [Carrd.co](https://carrd.co).

## Структура проекта

```
/carrd/
├── BREAKPOINTS.md          # Стандартные breakpoints для всех проектов
├── mini/                    # Проект 1
│   ├── src/                # Исходники (референс)
│   │   ├── index.html     # Финальная сборка страницы
│   │   └── assets/
│   │       ├── main.css   # Стили проекта
│   │       └── main.js    # Скрипты проекта
│   ├── grid-cluster.html   # Кастомизация 1
│   └── smooth-scroll.html  # Кастомизация 2
├── another-project/        # Проект 2
│   ├── src/
│   │   ├── index.html
│   │   └── assets/
│   └── custom-code.html
└── README.md
```

## Правила

- **Папки проектов** — содержат `index.html` (основной файл) и файлы кастомизаций
- **Кастомизация** — это отдельный `.html` файл в папке проекта
- Каждый файл содержит только `<style>` и `<script>` блоки
- Вверху файла — HTML комментарий с описанием функциональности
- Код готов для прямого копирования в Carrd
- **Breakpoints** — используйте стандартные breakpoints из `BREAKPOINTS.md`

## Breakpoints

Все проекты используют единые breakpoints (см. `BREAKPOINTS.md`):
- 1920px, 1680px, 1280px, 1024px, 980px, 736px, 480px, 360px

Конкретные значения для padding, font-size и layout зависят от дизайна каждого проекта.

## Использование

1. Откройте нужный `.html` файл в папке проекта
2. Скопируйте содержимое (комментарий + `<style>` + `<script>`)
3. Вставьте в кастомный код вашего Carrd сайта

## Создание новой кастомизации

Используйте workflow `/create-carrd-customization` для создания новых кодов.

Код будет создан в текущей папке проекта и проверен против `src/index.html`.

## Референсные файлы

- **HTML структура**: `{project}/src/index.html` — финальная сборка страницы
- **CSS паттерны**: `{project}/src/assets/main.css` — стили проекта
- **JS паттерны**: `{project}/src/assets/main.js` — скрипты проекта
