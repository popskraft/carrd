# Carrd Grid Cluster

Набор файлов для распространения кастомизации Grid Cluster через jsDelivr. Репозиторий содержит только то, что нужно для продакшн‑деплоя: минифицированный бандл и открытые исходники конкретной кастомизации.

## Назначение
- `src/mini/grid-cluster/` — исходный CSS и JavaScript, которые правятся вручную.
- `dist/mini/grid-cluster.{css,js}` — готовые файлы, которые подхватывает jsDelivr и вставляют на сайт Carrd.
- `mini/grid-cluster.html` — автогенерируемый сниппет для вставки в Carrd (получается из исходников при сборке).

## Как устроено
1. Исходные файлы правятся в `src/mini/grid-cluster/`.
2. Сборка (`npm run build`) минифицирует содержимое, обновляет `dist/`, актуализирует CDN manifest и пересобирает сниппет.
3. Репозиторий остаётся лёгким: помимо исходников видны только артефакты, которые реально нужны для встраивания.

## CDN
Файлы распространяются через jsDelivr. Актуальные ссылки всегда лежат в `dist/` и перечислены в `docs/CDN-LINKS.md`, который обновляется автоматически во время сборки.
