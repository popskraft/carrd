# Grid Cluster

## Описание
Автоматически группирует непрерывные блоки с классом `style-3` в контейнеры `custom-grid-container` с grid-раскладкой 3 колонки.

## Как работает
1. Скрипт находит все элементы с классом `style-3`
2. Определяет "кластеры" — непрерывные последовательности этих элементов
3. Если между блоками есть другой элемент, это разные кластеры
4. Каждый кластер оборачивается в `custom-grid-container`
5. CSS применяет grid: 3 колонки, gap 1rem

## Использование
1. Подключить `grid-cluster.js` перед закрытием `</body>`
2. Подключить `grid-cluster.css` в `<head>`
3. Скрипт автоматически срабатывает при загрузке страницы

## Особенности
- Не изменяет исходный HTML (только добавляет обертку)
- Работает с динамически добавленным контентом
- Минимальный размер кода
